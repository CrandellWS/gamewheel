# Full Tilt Mode - User Guide

## What is Full Tilt Mode?

**Full Tilt** is an exciting ladder-climbing game mode where players race to reach the top! Each spin of the wheel selects one player who climbs up one rung on the ladder. The first player to reach the top wins everything in a dramatic "all or nothing" showdown.

## Key Features

- **Ladder Climbing**: Players start at the bottom and climb one rung at a time
- **Configurable Heights**: Choose 3, 5, 7, or 10 rungs for different game lengths
- **Visual Progress**: Beautiful animated ladder display shows everyone's position
- **Dramatic Finale**: Confetti and celebration when someone reaches the top
- **Complete History**: Every ladder climb is tracked and exportable

   - **5 Rungs**: Standard game (5-20 spins typical) - **Recommended**
   - **7 Rungs**: Extended game (7-30 spins typical)
   - **10 Rungs**: Marathon game (10-50+ spins typical)

### 2. Add Your Players

Add players to the wheel using any method:
- Type names individually in the entry list
- Bulk add multiple names at once
- Import from a saved configuration
- Accept entries via chat integration (Twitch, Discord, YouTube)

### 3. Start Spinning!

1. Click **SPIN** or press **Spacebar**
2. Watch the wheel spin and select a player
3. The selected player climbs **one rung** on the ladder
4. Click **Confirm Winner** to proceed
5. Repeat until someone reaches the top!

## Understanding the Ladder Display

The ladder display appears below the wheel when Full Tilt mode is active.

### Ladder Components

#### Rungs (Numbered 1-10)
- **Top Rung** (Winning Position): Gold border, trophy icon üèÜ
- **Player Badges**: Show which players are on each rung
- **Empty Rungs**: Display "Empty" until a player reaches them

#### Waiting to Climb Section
- Shows all players who haven't started climbing yet
- Players disappear from here when they climb to rung 1
- Color-coded badges match wheel colors

#### Stats Panel
- **Climbing**: Number of players currently on the ladder
- **Highest Rung**: The furthest anyone has climbed so far

## Frequently Asked Questions

### Q: Can I change the ladder height mid-game?

**A:** Yes, but it's recommended to reset the wheel first. Changing mid-game won't affect players already on the ladder but will change the winning threshold.

### Q: What happens if I switch game modes with players on the ladder?

**A:** When you switch away from Full Tilt mode, ladder positions are preserved but the ladder display disappears. If you switch back, positions resume. Use "Reset Wheel" to start fresh.

### Q: Does the same player climb every time they're selected?

**A:** Yes! Each time a player is selected, they climb one additional rung. The first player to reach the top wins.

### Q: Can multiple players win?

**A:** No, Full Tilt is a "winner takes all" mode. Only the first player to reach the top rung wins. After they win, you can reset and start a new game.

### Q: How does this work with multi-winner selection?

**A:** In Full Tilt mode, even if you have multi-winner selection enabled (3, 4, or 8 winners), only the **primary winner** (top of wheel) climbs the ladder. This keeps the ladder mechanic simple and competitive.

### Q: Can I import/export Full Tilt games?

**A:** Yes! Full Tilt settings (ladder height) are included in wheel exports. However, current ladder positions are **not** exported - only the configuration.

## Troubleshooting

### Ladder display not showing
- Verify Full Tilt mode is selected in Settings
- Refresh the page if needed

### Players not climbing
- Make sure you clicked "Confirm Winner" after each spin
- Check browser console for errors

### History not tracking climbs
- Ensure you're in Full Tilt mode when spinning
- Export history to verify data is being recorded

### Performance issues
- With 20+ players, consider using a smaller ladder height
- Close other browser tabs for better performance
- Disable confetti if needed (Settings ‚Üí Confetti animation)
